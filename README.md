
# ai-fragrance-recommender
A beginner python application that recommends fragrances based on user preferences

## Overview
The AI Fragrance Recommender is a beginner Python project created as part of my journey into software development and artificial intelligence.

This application asks users about their fragrance preferences and recommends fragrance families based on their selection.

## Features
- User-friendly menu
- Four fragrance categories
- Instant fragrance family recommendations
- Beginner-friendly Python code

## Technologies Used
- Python

## Why I Built This Project
As the founder of L.N Fragrance, I want to combine technology and perfumery to create intelligent tools that help customers discover fragrances that match their preferences.

This project is my first step toward building AI-powered fragrance recommendation systems.

## Future Improvements
- Personalized fragrance recommendations
- Machine learning integration
- Customer preference database
- Web application interface
